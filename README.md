### ğŸ’¡ Tips for Success

**1. Understand Before Coding**
- Read the problem 2-3 times
- Draw examples on paper
- Identify the pattern before writing code

**2. Practice Active Recall**
- Try solving again after 1 day
- Then after 1 week
- Then after 1 month

**3. Time Yourself**
- Easy: 15-20 minutes
- Medium: 30-45 minutes
- Hard: 45-60+ minutes

**4. Learn from Solutions**
- If stuck for >30 mins, look at hints
- Study optimal solutions
- Implement from memory later

**5. Focus on Patterns**
- Don't memorize solutions
- Understand the underlying pattern
- Apply pattern to similar problems
